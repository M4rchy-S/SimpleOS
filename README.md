# SimpleOS
Custom OS kernel made for educational purpose.
<li>GRUB bootloader</li>
<li>Protected mode (32-bit)</li>
<li>GDT is set</li>
<li>IDT is set with keyboard ISR</li>
<li>Paging disabled</li>
<li>Driver to RTC module</li>
